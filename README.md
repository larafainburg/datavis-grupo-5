Narrativa Visual Interactiva

Este proyecto consiste en una narrativa visual interactiva desarrollada para web, diseñada para ofrecer a los usuarios una experiencia inmersiva sobre los distintos países visitados por los integrantes del grupo.

A medida que los lectores avanzan haciendo scroll, descubren una serie de visualizaciones interactivas que presentan diferentes aspectos de la investigación, explorando dimensiones como:

* Localización geográfica
* Años y períodos de viaje
* Categorías de análisis
* Magnitudes y comparaciones
* Distribución y diversidad de destinos

El objetivo principal del proyecto es transformar datos e información en una experiencia visual dinámica y atractiva, combinando storytelling, diseño interactivo y visualización de datos para facilitar la exploración y comprensión de los contenidos.

Tecnologías utilizadas

* HTML / CSS / JavaScript
* Visualización de datos interactiva
* Diseño responsive
* Herramientas de desarrollo web

Características

* Experiencia basada en scroll interactivo
* Visualizaciones dinámicas
* Narrativa visual enfocada en exploración de datos
* Diseño centrado en experiencia de usuario
* Adaptación a distintos dispositivos
